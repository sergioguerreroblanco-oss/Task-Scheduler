Detailed Description

Implementation of the thread-safe job queue.

Date

Version
1.0.0

This file contains the full implementation of JobQueue, a multi-producer multi-consumer FIFO queue used as the backbone of the thread pool.

Key properties:

  • Safe concurrent access through std::mutex.
  • Blocking pop with condition variable.
  • Deterministic shutdown: waiting consumers wake up and return nullptr.
  • Ownership is transferred using std::unique_ptr<IJob>.
