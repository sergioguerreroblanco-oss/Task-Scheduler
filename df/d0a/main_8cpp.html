Detailed Description

Entry point for the Task Scheduler demo application.

Version

This program demonstrates:

  • Starting a ThreadPool with configurable number of threads
  • Enqueuing jobs (PrintJob)
  • Graceful shutdown()
  • Immediate shutdownNow()

CLI options: –threads N Number of threads (default: HW concurrency) –demo Enqueue several PrintJobs –slow Enqueue slow jobs (requires FakeSlowJob) –immediate-shutdown Stop immediately (shutdownNow)
