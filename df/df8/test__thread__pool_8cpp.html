Workers must survive exceptions thrown by jobs.

Test:

GIVEN a thread pool with 1 worker AND a FakeThrowingJob followed by a normal FakeJob WHEN both are enqueued ,executed and after that the thread pool is shutted down THEN:

  • The throwing job must mark executed == true
  • The next job must still be executed (worker is still alive)
  • No crash must occur
