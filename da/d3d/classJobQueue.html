◆ shutdown()

void JobQueue::shutdown ( )

Shutdown the queue, unblocking all waiting threads.

Sets an internal flag (closed = true) and notifies all waiting threads so they can exit gracefully.

After calling this, subsequent calls to pop() will return false once the queue is empty.
